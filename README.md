# Image_Processing_Api
## image processing with node js using express

**Backend**


    - Node.js
    - Typescript
    - Sharp package

**Tooling**

    - ESLint
    - Nodemon
    - Jasmine
    - Prettier


**Scripts**

### Installing 
    npm install

### Build  
    npm run build

### EsLint  
    npm run lint

### Prettier :

    npm run format

### Run unit tests  
    npm run test

### Start server
    npm run start



## Endpoint to resize images

> [http://localhost:3000/api](http://localhost:3000/api)


### Expected filenames

    - icelandwaterfall
    - encenadaport
    - santamonica
    - fjord
    - palmtunnel


### Expected width and hieght

    - 0< hieght < 10000
    - 0< width  < 10000


### valid example

> http://localhost:3000/api?name=encenadaport&height=1000&width=1000
